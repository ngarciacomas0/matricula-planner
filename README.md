# matricula-planner
Una aplicación de python que genera horarios universitarios válidos con materia, grupos de horas, profesores y tiempo
## Funcionalidades
* Añadir, modificar y eliminar materias y grupos de horas
* Almacenar grupos con profesores y NRC (Número de Referencia de Curso)
* Seleccionar grupos preferidos
* Eliminar horarios con conflictos automáticamente
* Generar todos los horarios válidos
* Filtrar horarios con preferencias
* Ordenar horarios por:
     * Balanceo de horas (horas uniformemente repartidas en la semana)
     * Mayores horas en la mañana / tarde
     * Horas en un rango personalizado
     * Profesores preferidos
* Exportar horarios generados a Excel
